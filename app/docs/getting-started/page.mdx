# 开始使用

欢迎使用我们的文档系统！本指南将帮助您快速上手使用各项功能。

## 特性

- 📝 支持 Markdown 和 MDX
- 🔍 全文搜索
- 📤 文件上传
- 🎨 美观的界面
- 📱 响应式设计

## 快速开始

1. **浏览文档**
   - 使用左侧导航栏浏览不同章节
   - 使用顶部搜索框查找内容

2. **上传文档**
   - 点击"上传"按钮
   - 选择要上传的文件
   - 等待上传完成

3. **编辑文档**
   - 使用内置的 Markdown 编辑器
   - 实时预览编辑效果
   - 一键保存更改
